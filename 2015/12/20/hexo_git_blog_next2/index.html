hexo搭建github博客(NexT主题)(二)

一. 编写并发布博文(以macdown为例)

  1. 打开macdown,commadn+N,填写标题,分类,标签信息,日期可选 

    title: hexo博客布局及文章发布
date: 2015-12-20 11:00:00 
categories: 软件及网络应用
tags: [hexo,NexT,blog,教程] 

---

  2. 编写博文

  3. 终端进入本 地博客目录 

    $ cd ~/Develop/yangzongjie.github.io

  4. 部署至github 

    $ hexo g -d

    每一行尾至少接2个空格,回车后才能换行 (关于markdown语法,可参考macdown的help文件)

二 hexo主要文件系统

  1. /yangzongjie.github.io/_config.yml 站点配置文件
  2. /yangzongjie.github.io/theme/_config.yml主题配置文件
  3. /yangzongjie.github.io/source/_posts 博文存放路径(.md,推荐用macdown或mou编写)

三. 设置博客外观(头像,栏目,社交链接,评论等第三方服务)

  1. 设置方法参考 NexT官方文档
    http://theme-next.iissnan.com/five-minutes-setup.html
    http://theme-next.iissnan.com/theme-settings.html

  2. icon及名称目录:https://icomoon.io/#preview-essential

  3. 显示”阅读更多”,在相应文本处添加<!-- more -->.

四. 文章插入图片|视频

  1. 插入本地图片(md语法): ![](/images/**.png)

    图片保存路径:/yangzongjie.github.io/source/images,需手动创建.

  2. 插入视频

  • 方法1: html5播放器

  • 方法2: 链接youku,b站视频分享中的html地址.
